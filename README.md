This is a small set of code and sub actions in streamerbot that takes a text document as input, makes an array, then converts it into a list.
That list is then altered based on what category you are streaming in so there is never any overlap.
Iterates over those elements to generate global variables that are then used in a poll.
The poll runs, and based on the result automatically changes your stream category/title.
It also runs every hour. The main idea is to offer the community an engaging way to change it up with what game we are playing.
It's also set up so that if a game is already being played, the first option of the poll is always "Keep Playing This", because we wouldn't to FORCE anyone to switch, that would be sad!
